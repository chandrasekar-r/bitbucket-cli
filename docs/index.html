Take Bitbucket

What bb can do

Every Bitbucket workflow,
from your terminal.

Pull Requests
Create PRs with auto-generated descriptions from commit history and optional PR templates. Edit after creation with an interactive form or flags.
$ bb pr create
✓ Template: .bitbucket/PULL_REQUEST_TEMPLATE.md
✓ Jira: PROJ-456 detected on branch
✓ Created PR #89: Auth middleware refactor
https://bitbucket.org/myteam/backend/pull-requests/89
Pipelines
Watch live log output and get a native desktop notification when the build finishes.
$ bb pipeline watch --notify
Pipeline #142 started
[step 2/3] Run tests ✓
[step 3/3] Build & push ✓
SUCCESSFUL (2m 14s) 🔔
Status Dashboard
Your morning standup, without opening a browser.
$ bb status
Your Open Pull Requests
89 backend  Auth middleware 2✓
Awaiting Your Review
91 backend  Rate limiting (alice)
Branch Tidy
Delete local branches whose PRs are merged or declined. Never clean up by hand again.
$ bb branch tidy --dry-run
Merged:
  feat/oauth-refactor PR #82
  fix/login-validation PR #79
Would delete 2 branch(es).
Raw API
Authenticated access to any Bitbucket endpoint. Pagination, jq filtering, nested field construction. The ultimate escape hatch for anything not yet in bb.
$ bb api /repositories/{workspace} --paginate --jq '.[].full_name'
"myteam/backend"
"myteam/frontend"
"myteam/shared-lib"
$ bb api /user --jq '.username'
"rc"
Batch Operations
Run any command across every repo in a workspace with configurable concurrency.
$ bb batch --repos "backend-*" --
    pipeline list --limit 3
── myteam/backend-api ────
#141 SUCCESSFUL  main  2m
── myteam/backend-auth ───
#98   SUCCESSFUL  main  1m

No context switching

Review PRs, watch pipelines, file issues, manage repos — all from the terminal. Stay in the flow.

First-class scripting

Every command supports --json and --jq. bb api reaches any endpoint. Pipe into anything.

Desktop notifications

Run bb pipeline watch --notify, switch tasks, and get pinged when the build finishes.

Interactive & headless

Smart pickers and forms in TTY. Strict --no-tty mode for CI/CD. Works everywhere, adapts automatically.

Tab completion

PR numbers, branch names, and repo slugs complete live from the API. Bash, zsh, fish, PowerShell.

Extensible

Install community plugins with bb extension install. Any git repo with an executable becomes a command.

Installation

Get bb in seconds.

macOS — Homebrew
brew install chandrasekar-r/tap/bb
Windows — Scoop
scoop bucket add chandrasekar-r https://github.com/chandrasekar-r/scoop-bucket
scoop install bb
Linux — Debian / Ubuntu
curl -Lo /tmp/bb.deb https://github.com/chandrasekar-r/bitbucket-cli/releases/latest/download/bb_linux_amd64.deb && sudo dpkg -i /tmp/bb.deb
Universal — curl installer
curl -fsSL https://raw.githubusercontent.com/chandrasekar-r/bitbucket-cli/main/scripts/install.sh | sh

Need a different platform? Download binaries directly → for Linux arm64, rpm, macOS universal, and Windows amd64.
