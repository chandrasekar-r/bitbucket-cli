## Key Decisions Made During Build

| Decision                                  | Rationale                                                                                         |
| ----------------------------------------- | ------------------------------------------------------------------------------------------------- |
| Go language                               | Single static binary, industry standard for CLIs (`gh`, `kubectl`)                                |
| `bb` binary name                          | Mirrors `gh`, natural Bitbucket abbreviation                                                      |
| File-lock on token refresh                | Bitbucket rotating refresh tokens enforced May 4, 2026 — prevents concurrent session invalidation |
| No app password support                   | Deprecated June 9, 2026                                                                           |
| Polling (not streaming) for pipeline logs | Bitbucket Pipelines has no SSE/WebSocket endpoint                                                 |
| `errcheck` partially disabled             | `fmt.Print*`, `defer Close/Unlock` errors are not actionable in CLI tools                         |
| golangci-lint v2 (not v1)                 | Required for Go 1.25+ module support                                                              |
